# Linux 学习笔记

## 在 CentOS 8 上安装和使用 Apache Web 服务器

### 安装

```
sudo yum install httpd
```

### 启动

```
sudo systemctl start httpd
```

### 查看状态
```
sudo systemctl status httpd
```

### 停止
```
sudo systemctl stop httpd
```

参考：[如何在CentOS 7上安装Apache Web服务器
